== Description ==
A custom theme for displaying projects and a contact form

== Installation ==
1. Download the theme from the WordPress repository or upload it via the theme uploader.
2. Activate the theme through the "Appearance > Themes" menu in WordPress.
3. Customize your settings using the WordPress customizer and theme options.

== Theme Features ==
* Custom Post Types: Projects with custom fields.
* Responsive layout.
* Full support for the block editor (Gutenberg).
* Customizable homepage with hero section and services.
